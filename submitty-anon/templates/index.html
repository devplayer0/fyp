Beyond functional autograding (Final year project)

This project seeks to explore the potential for grading assignments and providing insights based on code performance. To test the effectiveness of the system, there is a temporary Submitty installation which will allow for submission of prior Introduction to Computing II assignments. The system will provide a performance grade as well as some graphs and data. Feel free to experiment with your code (the grading curves may be far off, even for an efficient program!).

For more details, see the Information sheet.

Your code will be evaluated a number of times in both a simulator and real hardware (pictured above!)
